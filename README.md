<h1>Terminal N</h1>

<p>A really easy to use terminal tool that:</p>
<ul>
	<li>Creates notes</li>
	<li>Modifies Notes</li>
	<li>Deletes notes</li>
	<li>Creates encrypted notes with AES</li>
	<li>Deciphers encrypted notes</li>
</ul>

<p>Requirements</p>
<ul>
	<li>python == 3.5.2</li>
	<li>pycrypto == 2.6.1</li>
</ul>
